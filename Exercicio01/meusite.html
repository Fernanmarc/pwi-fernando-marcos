Configurando o Git Bash no PC

Registrando seu GitHub no Git Bash

Após ter o programa em seu computador é necessário configurar para poder começar a programar.
E para isso, tem determinados passos que devem ser seguidos:

  1. Tenha o Git Bash instalado e aberto em seu computador.
  2. Pesquise no google 'configuração inicial git' ou entre nesse link para realizar a configuração
  3. Copie os codigos em ordem e cole no seu Git Bash
  4. Primeiro codigo user.name (trocando o username pelo seu nome do github), em segundo a linha do codigo do email (trocando agora seu email utilizado na conta do github).

Clonando seu repositório do GitHub

Configurando seu github no seu git bash, o proximo passo é pegar sseu repositório!
Vamos lá:

  1. Entre no seu GitHub (Com seu Login e senha usando no registro do Git Bash).
  2. No topo da pagina, clique na sua foto e depois em repositório.
  3. Escolha a pasta que você deseja usar, clique no code. Após isso, aparecerá uma mensagem, escolha a opção HTTPS e clique em copiar.
  4. Com o link, abra o Git Bash e escreva "Git Clone + link" e de enter, após isso os arquivos serão clonados para sua área de trabalho.
